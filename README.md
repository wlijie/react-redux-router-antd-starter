## Getting Started

Just clone the repo and install the necessary node modules:

```shell
$ git clone https://github.com/wlijie/react-redux-router-antd-starter.git demo
$ cd demo
$ npm install
$ npm start
```

## Build

```shell
$ npm run build
```
