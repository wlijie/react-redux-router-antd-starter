## react-redux-router-antd-starter 安装使用

## Install

```bash
$ npm i antd-init -g
```

## Usage

Generate demo boilerplate.

```bash
$ mkdir foo && cd foo
$ antd-init
```

Start development server.

```bash
$ npm start
```

Build.

```bash
$ npm run build
